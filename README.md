# TeamFlow-Assignment1

TeamFlow is a responsive SaaS website built with HTML, CSS, and JavaScript to manage teams, projects, and tasks.It includes a landing page, product features, blog section, and simple pricing plans laid out with CSS Grid and Flexbox.Reusable components, modular CSS, and vanilla JavaScript provide clean interactions like alerts without external libraries.The design follows modern SaaS UI patterns, highlighting solid front-end fundamentals suitable for real-world use and portfolios.
